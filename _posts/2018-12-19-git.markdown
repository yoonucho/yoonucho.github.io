## 커밋 후 커밋 메시지 변경 (push전)

1.  git commit --amend
2.  insert 버튼을 눌러서 메세지 수정후 ESC
3.  :wq로 저장+종료

## 커밋 후 커밋 메시지 변경 (push후)

1.  git commit --amend
2.  insert 버튼을 눌러서 메세지 수정후 ESC버튼을 눌러줌
3.  :wq로 저장+종료
4.  git push -f(force)

이렇게 하면 이쁜(?) 일직선 형태의 log를 볼 수 있음.

## add후 취소

- git reset HEAD [파일명]
  ex) index.html를 add후 취소
  - git reset HEAD index.html
